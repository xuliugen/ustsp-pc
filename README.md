# USTSP
> University Science and Technology Service Platform

## 准备
- Node.js
- [yarn](https://yarnpkg.com/zh-Hans/docs/install)

## 开发
``` bash
# 安装依赖
$ yarn

# 开发
$ yarn run start
```

## 文档
[开发文档](/doc/dev.md)
[规范](/doc/standard.md)

## 项目结构
`/src/views/` 中的组件为大的页面组件，`/src/components/` 中的组件为页面组件中拆分出来的小组件。
